### [0.2.6] - 2026-06-27

#### Added
- **新精灵 26 只**：共计 111 种，新增以下进化线与独立精灵
  - 火猫系：祥云火猫 → 烈云猫兽 → 焱纹虎（火/风）
  - 铎狐系：铎狐（钢/灵）→ 钺霄（钢/风）
  - 蛛系：蛛优优 → 蛛灵儿 → 盘丝妖后（虫/毒，西游题材）
  - 猴系：小顽猴 → 持棍猿 → 猿圣（格→格/龙，齐天大圣题材）
  - 幽魂系：幽荧 → 夜幽魂 → 墟梦魇（鬼/暗）
  - 铜角系：铜角幼 → 战甲铜（灵） → 蚩极（格/暗，地品）
  - 独立：大树鲁特（木/灵，玄，灵芽儿终态）
- **三圣兽**：炎凰（火/仙 神）/ 霆啸（雷 地）/ 玄溟（水/暗 地），三者并列神兽级
- **矛盾双煞**（上版遗留）：寒角 → 凌矛犀（冰/土）/ 铁鳞 → 风盾甲（风/钢）
- **设计规范**：新增 `design_guide.md`，统一生图 Prompt 后缀、属性字根表、进化30/70法则

#### Fixed
- **双属性克制计算 bug**：两个 STAB 属性对同一防御属性冲突时（一克一抵），正确从两侧列表移除，不再重复显示
- **新建精灵 ID 逻辑**：从最小未使用 ID 开始分配，避免跳过 85-98 区间直接到 178
- **蛛系链修正**：去除不存在的"小蛛妖"，蛛优优直接进化蛛灵儿
- **大树鲁特链修正**：灵芽儿终态为大树鲁特（古檀灵归属核胄叶线，待素材加入）
- **霆啸定位修正**：由普通玄级改为三圣兽地品（BST 610），与炎凰/玄溟并列
