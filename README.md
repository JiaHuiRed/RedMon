## 游戏简介

**红灵**（RedMon）是一款致敬宝可梦火红/叶绿的中国风像素 RPG。

玩家扮演来自华灵大陆的训练师，从陈教授处获得第一只精灵，开始探索、捕捉、对战的旅途。

---

## 核心特性

- **御三家选择** — 炎喵（火）、蓝蛇（水）、小竹熊（木），由陈教授赠予
- **回合制战斗** — 火红风格 UI，速度先攻，4 技能选择，含背包/换宠/逃跑
- **多场景流程** — 角色创建 → 选御三家 → 家（妈妈对话）→ 青木村 → 劲敌战 → 华灵草原 → 碧溪镇 → 翠竹馆
- **19 属性克制** — 基于宝可梦第六世代属性表 + 原创光属性，汉化为中文属性名
- **种族值系统** — 6 维属性（HP/攻击/防御/特攻/特防/速度），含 IV 个体值
- **性格系统** — 25 种性格（`data/natures.json`），影响属性 ±5% 成长
- **特性系统** — 每只精灵天生携带被动特性（主特性/隐藏特性），影响战斗策略
- **食疗式努力值** — 通过滋补食物（中国食补主题）提升精灵属性，取代传统刷怪练努力值
- **状态异常** — 烧伤、中毒、麻痹、睡眠、冰冻，含回合效果
- **经验与升级** — 3 种成长速度（快速/中速/缓慢），升级自动习得技能
- **进化系统** — 等级进化 + 道具分支进化（如露比 5 条分支），战斗中弹出选择
- **捕捉系统** — 精灵葫芦投掷动画，基于 HP、状态、种族捕获率的概率公式
- **道具背包** — 精灵葫芦、回复丹药、进化道具、滋补食物等 35 种道具
- **键盘全套映射** — Z 确认 / X 取消 / Enter 菜单 / Space 跑步，Godot input map 统一管理
- **多精灵队伍** — 最多 6 只，战斗中可切换，被击倒后强制换场，PC 仓库存储
- **道馆系统** — 翠竹馆（草系 1 号馆）已实装，含杂兵训练师 + 馆主战
- **NPC 系统** — 林薇（跑步鞋 + 捕捉奖励）、训练师对战、商店、灵疗所
- **3 档存档** — 标题画面存档槽选择，随时存读
- **可视化编辑器** — 内置 tkinter 编辑工具，可视化修改精灵/技能/道具/训练师/剧情文本/特性库

> 完整数据见 `data/species.json`，可通过 `tools/mon_editor.py` 可视化浏览和编辑。

| 名称 | 属性 | 特点 | 进化路线 |
|------|------|------|----------|

| 蓝蛇 | 水 | 防御型，通体蔚蓝的小水蛇 | 蓝蛇 Lv16 → 江蛟 Lv36 → 覆海龙 |

## 操作说明

### 键盘（GBA 风格映射）

| 游戏按键 | 键盘 | 功能 |
|----------|------|------|
| A 按钮 | `Z` / `Enter` | 确认 / 与 NPC·建筑交互 / 推进对话 |
| B 按钮 | `X` / `Esc` | 取消 / 返回上级菜单 / 呼出/关闭主菜单 |
| D-Pad | 方向键 | 大地图移动 / 菜单光标导航 |

### 手柄（Godot 默认映射，即插即用）

| 手柄按钮 | 功能 |
|----------|------|
| A | 确认 |
| B | 取消 / 返回 |
| 左摇杆 / 十字键 | 移动 / 导航 |

### 大地图

| 操作 | 说明 |
|------|------|
| 方向键 | 移动角色 |
| `Z` / Enter | 与面前建筑/NPC 交互；推进对话 |
| `X` / Esc | 呼出主菜单（精灵 · 背包 · 存档 · 关闭） |
| 主菜单内 `X` / Esc | 返回上一级 / 关闭菜单 |
| 进入灵疗所 | 全队 HP/PP 回满并自动存档 |
| 进入道具店 | 上下选商品，Enter 购买，Esc 离开 |

### 战斗

| 操作 | 说明 |
|------|------|
| 方向键 | 在行动/招式面板间移动光标 |
| `Z` / Enter | 确认当前选项 |
| `X` / Esc | 返回上一面板（招式→行动） |
| 鼠标点击 | 与键盘等效，可混用 |

**行动面板（2×2）**


## 编辑器

项目附带可视化编辑器 `tools/mon_editor.py`（tkinter），支持：

- **精灵图鉴**：种族值条形图（含 Lv50/Lv100 预估）、BST 色阶、属性克制面板、遭遇地编辑
- **属性/性别/体型/描述**：可视化编辑所有字段
- **特性分配**：主特性/隐藏特性下拉框，支持搜索过滤
- **图鉴预览**：右侧实时显示正面 + 背面 PNG 精灵图
- **进化链对比**：全链 BST 条形图横排对比，支持分支进化编辑（道具/等级）
- **技能池**：支持添加、删除、编辑习得等级；技能名支持搜索
- **技能库**：技能增删改，技能名/属性/分类/威力/效果类型/概率/数值全字段
- **特性库**：特性增删改，名称/描述/效果键编辑
- **道具库**：道具增删改，名称/类别/价格/描述全字段
- **角色编辑**：训练师数据（队伍、奖励、对话）
- **剧情文本**：对话文本编辑（中文标签），支持字符串/数组/嵌套字典
- **数据持久化**：直接读写 species.json / moves.json / items.json / trainers.json / dialogs.json / abilities.json

#### 编辑器快捷操作

| 操作 | 说明 |
|------|------|
| 左侧列表搜索框 | 输入精灵名实时过滤（支持属性/品阶筛选） |
| 点击列表条目 | 加载精灵数据 |
| 点击保存 | 写入 species.json |
| 技能池 → + 添加技能 | 弹窗搜索技能名，填写习得等级 |
| 技能池 → 编辑等级 | 修改选中技能的习得等级 |
| 技能池 → 删除技能 | 移除选中行 |

## 运行方法

> **本项目使用 Git LFS 管理图片资源，克隆前必须先安装 git-lfs，否则精灵图会变成空文件。**

### 第一步：安装 Git LFS（仅需一次）

- **Windows**：前往 [git-lfs.github.com](https://git-lfs.github.com/) 下载安装包
- **macOS**：`brew install git-lfs`
- **Linux**：`sudo apt install git-lfs`

安装后运行：
```bash
git lfs install
```

### 第二步：克隆并运行

1. 安装 [Godot 4.7](https://godotengine.org/download/)
2. 克隆本仓库（LFS 文件会自动下载）：
   ```bash
   git clone <repo-url>
   ```
3. 若图片未自动下载，手动拉取：
   ```bash
   git lfs pull
   ```
4. 在 Godot 中打开 `RPG_Demo/project.godot`
5. 点击运行（F5）

### 运行编辑器

```bash
python -X utf8 tools/mon_editor.py
```

### 添加自定义精灵图片

将 PNG 文件放入 `assets/sprites/`，命名格式：
- `{精灵名}front.png` — 敌方（正面）
- `{精灵名}back.png` — 我方（背面）

例：`炎喵front.png`，`炎喵back.png`

## 属性克制表（19属性）

```
空 火 水 木 雷 冰 格 毒 土 风 灵 虫 岩 鬼 龙 暗 钢 仙 光
```

光属性克制关系：光克鬼/虫/冰/木/暗，光被土/钢/暗克，光打火/钢/光/水效果不佳，光↔暗互相克制。

完整克制关系见 `scripts/autoload/mon_db.gd`（type_chart 字典）。
