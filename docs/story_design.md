# RedMon（红灵）剧情大纲
> 本文件是游戏唯一的剧情总源文档。所有场景脚本/对话/NPC设计以此为准。

---

## 一、世界观

- **大陆名称**：华灵大陆
- **风格**：东方奇幻，人与精灵共存，山河大地皆有灵气
- **货币**：灵币（G）
- **属性体系**：共 19 种属性（18 标准 + 光系）
- **图鉴目标**：收录华灵大陆全部 305 种精灵

## 四、重要 NPC

### 陈教授（陈旭敏）
- **身份**：华灵大陆精灵研究权威
- **立绘**：博士front.png
- **登场**：序章开场白 + 御三家选择场景
- **台词风格**：热情、话多、有点夸张的老学者
- **主线作用**：给予御三家 + 图鉴；后期拥有重要研究信息

### 林薇
- **身份**：陈教授助理
- **立绘**：待制作
- **登场**：青木村，给跑步鞋；每捕获 10 只精灵奖励 1000G
- **性格**：温柔活泼，是玩家在村里的"指引 NPC"

### 申鹤
- **身份**：神秘强者，独行者，来历不明
- **立绘**：申鹤walk_sheet.png / 申鹤front.png
- **属性**：待定（偏向灵/冰风格）
- **对战队伍**：坤仔 Lv14 + 炎喵 Lv16
- **性格**：冷漠孤高，嘴硬心软，不屑路人但总在关键时刻出现
- **三段出场**（已实装，注意：实际登场地点是华灵草原入口而非青木村内，青木村内没有申鹤的内容）：
  1. 华灵草原入口：嘲讽主角没精灵，顺口提示黑风堂在草原出没
  2. 华灵草原出口：提示黑风堂在镇里有眼线，预告林青松战术
  3. 碧溪镇道馆前：强制对战，"考考你有没有资格进道馆"
- **剧情定位**：类似劲敌——阶段性出现、对战、推进，暂无独立支线
- **终局身份**：计划担任**仙道天王（仙系·第二代）**，游戏通关后可在冠军之路再战

## 五、反派——黑风堂

### 组织概况
| 项目 | 内容 |
|------|------|
| 名称 | 黑风堂 |
| 目标 | 控制各地灵脉 + 盗取/垄断稀有精灵 + 寻找远古龙灵 |
| 制服 | 黑色为主，带"风"字徽记 |
| 势力分布 | 遍布华灵大陆各区，各分部专司一类属性精灵 |

### 首脑

| 角色 | 说明 |
|------|------|
| **邬君梅** | 黑风堂堂主，龙系。**前期伪装成"美美"**（活泼的普通女训练师），青木村后山初遇玩家后，在华灵草原/碧溪镇等地多次相遇，逐渐建立"朋友"关系。中期某关键节点揭露真实身份，成为主要 BOSS。|
| **美美（伪装态）** | 看似热情的女训练师，实为邬君梅变装。对战时队伍弱（前期不露马脚），台词轻松搞笑。 |
| **黑化动机（幕后设定，不对玩家明说）** | 美美曾是真心疼惜精灵的训练师，在到处救助被人类弃养、逼疯的精灵过程中屡屡碰壁——救得了一只，救不了一整个大陆。渐渐走向极端：与其感化人类，不如用力量把稀有精灵"保护"起来，不再交还人类手中。以此为初心创立黑风堂，但组织建立后行事逐渐偏离她最初的善意（控制灵脉、盗取精灵），她自己也没意识到已经黑化。青木村后山初遇（发狂幽狐事件）是这条动机线的第一次埋伏笔，之后每次偶遇可逐步加深，"真身揭露"那场戏再完整回收。 |

## 八、主线剧情大纲

### 序章：出发前（已实装）
```
开场 → 陈教授开场白（介绍华灵大陆）
     → 性别选择 + 取名
     → 草原场景：野生精灵偷袭陈教授
     → 主角选择御三家（炎喵/蓝蛇/小竹熊）驱走野生精灵
     → 获得图鉴
     → 回家：妈妈送行，爸爸鼓励，妈妈提起劲敌约战
     → 出门
```

### 第一章：青木村（已实装）
```
青木村
├── 陈教授草丛遇袭事件：拿到蓝秋秋后、领御三家前，帮教授打退绿肥虫，正式选定御三家
├── 林薇：给跑步鞋 + 捕获奖励
├── 劲敌首战：村内约战（克制御三家）
├── 调皮小孩大毛：村内训练师
├── 头目战：君美（明雷野生精灵，高个体值）+ 小霞（旁观NPC，蛋奖励）
└── 村北支线·后山小径（260722 新增，进行中）
    ├── 采药人：氛围NPC，头目战前后台词不同
    ├── 发狂精灵头目战：幽狐（明雷野生精灵，高个体值，可击败拿蛋或直接捕捉带走）
    └── 美美（邬君梅伪装）在此初登场：旁观头目战，战后与玩家讨论人与精灵的相处方式，
        为其黑化动机埋下第一颗伏笔（幕后设定见"反派——黑风堂"一节）
```
