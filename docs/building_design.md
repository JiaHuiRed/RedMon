# RedMon 建筑设计规范

## 核心原则

1. **中式风格** — 整体走中式/和风像素风，青瓦白墙/灰檐素脊，非西式非日式城堡
2. **等轴俯视角（室内）** — 所有室内场景必须是 RPG Maker / GBA 宝可梦经典等轴俯视（三面墙围合 + 清晰可见的地板和家具顶面），**严禁正面墙直视角/壁纸式构图**
3. **葫芦代替精灵球** — RedMon 世界观中用精灵葫芦（棕色传统中式葫芦，下圆上窄带短柄）代替精灵球，所有场景中出现"球状捕获道具"时必须是葫芦
4. **尺寸分层** — 建筑从普通到气派分三档，每档"占地/层高/装饰量"有明确区分

---

## 一、建筑外观分级

### 第1档：普通民居（玩家家 / NPC小屋）
- 占地：1格（约32×32像素在TileMap中对应）
- 层数：1层
- 外观：灰瓦白墙，深褐木门窗，无多余装饰
- 门：普通单扇木门
- 窗：1-2扇小方格窗
- 典型：玩家家、NPC普通小屋

**提示词模板：**
> 纯白背景，居中完整建筑，GBA像素风宝可梦游戏建筑素材。
> 中式普通单层民居，灰瓦坡屋顶，素白墙，深褐木门木窗。
> 门前两级台阶。左右对称。气质朴素温馨。

### 第2档：特色建筑（杂货铺 / 小商店）
- 占地：1格（略宽于民居）
- 层数：1层（或带阁楼）
- 外观：灰瓦白墙，额外装饰元素
  - 杂货铺：门口挂布幌子（"商"字）、柜台、商品陈列
- 门：双开木门或推拉门
- 窗：大窗展示商品
- 典型：杂货铺、道具店

**提示词模板：**
> 纯白背景，居中完整建筑，GBA像素风宝可梦游戏建筑素材。
> 中式单层商铺，灰瓦坡屋顶，素白墙，深褐木框架。
> [店铺特色描述：布幌子/商品陈列/招牌]。
> 门前两层台阶。气质市井温暖。

### 第3档：大型建筑（研究所 / 道馆 / 精灵堂）
- 占地：2倍于普通民居
- 层数：2层
- 外观：
  - 歇山顶或重檐，青灰色瓦，屋脊平直简洁无夸张翘角
  - 素白墙体 + 深褐木框架
  - 门：大型双开高挑木门
  - 窗：一层落地窗/二层横窗（3-4扇）
  - 台阶：三层以上青石台阶
  - 装饰：石灯笼/石狮（非纸灯笼）
- 左右对称，庄重气派

**提示词模板：**
> 纯白背景，居中完整建筑，GBA像素风宝可梦游戏建筑素材。
> 中式大型[建筑类型]，宽约为普通房屋2倍，二层楼高。
> [屋顶描述]青灰色歇山顶，屋脊平直简洁，檐下无繁复斗拱。
> 墙体素白，深褐木框架。一层正中两扇高挑木门，[门两侧描述]。
> [各层窗描述]。
> [台阶和装饰描述]。
> 左右对称，气质[庄重/庄严/学术/神圣]。

## 二、室内场景视角规范（CRITICAL）

### 视图要求
- **视角**：等轴俯视角（GBA 宝可梦/ RPG Maker 经典视角）
  - 能看到地板的全貌
  - 能看到家具的顶面和台面
  - 看到的是家具"上面"而非"前面"
  - 三面墙围合（左墙 + 后墙 + 右墙），墙在画面边缘
- **反例（禁止）**：
  - 正面墙直视角（像壁纸/挂画）
  - 竖屏肖像构图
  - 只看到家具正面看不到台面
  - 透视过强导致看不清家具布局

### 提示词关键措辞

必须包含：
```
GBA像素风RPG室内场景，等轴俯视角（能清晰看到地板和家具顶面），
经典三面墙围合。

地板：[材质描述]
后墙（远处）：[后墙上的窗户/书架/柜子]
左墙（左侧面）：[左侧家具]
右墙（右侧面）：[右侧家具]
大厅中央：[中央展台/装饰]
天花板/灯光：[灯光描述]
整体色调：[色调描述]
```

### 不同档次室内

| 档次 | 空间大小 | 家具密度 | 装饰细节 | 示例 |
|------|---------|---------|---------|------|
| 普通（家） | 房间级 | 适中 | 沙发/茶几/地毯/花 | 家中、卧室 |
| 中等（商店） | 房间级稍大 | 密集 | 货架/柜台/商品展示 | 杂货铺内 |
| 大型（大厅） | 宽×2 | 疏密有致 | 书架墙/实验台/标本柜/中央展台 | 研究所内、道馆大厅 |

## 三、葫芦规范

- **精灵葫芦**：棕色传统中式葫芦造型，下圆上窄，有短柄
  - 无额外纹饰/图案
  - 朴素木色（暖棕渐变）
  - 可加红绳系颈
  - 尺寸：小（道具图标约16×16）、中（手持约24×32）、大（展示用）
- **其他葫芦变体**（名称变化，造型仍为葫芦）：
  - 超级葫芦（深棕/铜色装饰）
  - 高级葫芦（暗红/金色装饰）
  - 大师葫芦（深紫/金纹装饰）
  - 豪华葫芦（金边/玉饰）

提示词中涉及捕获道具时，统一描述为：
```
棕色传统中式葫芦（下圆上窄带短柄），非精灵球非圆球状。
```

## 四、生图质量门禁

每次生成建筑/室内图后自检：
- [ ] 室内是等轴俯视视角？还是正面墙视角？（禁止）
- [ ] 空间大小符合对应档次？（第3档 ≥2倍普通房间）
- [ ] 所有捕获道具是葫芦不是精灵球？
- [ ] 建筑外观是中式风格？不是日式/西式？
- [ ] 提示词中的所有"非以下内容"被尊重？
